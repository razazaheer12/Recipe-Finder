<h1 align="center">🍽️ Recipe Finder</h1>

<p align="center">
  A beautiful, feature-rich web application to discover delicious recipes from around the world. 
  Search for your favorite meals, view detailed instructions, check ingredients, and watch cooking videos on YouTube.
</p>

---

## ✨ Features

- 🔍 **Smart Search** — Search meals by name or keyword
- 📋 **Detailed Recipes** — View complete cooking instructions
- 🥗 **Ingredients List** — See all required ingredients with measurements
- 🎥 **Video Tutorials** — Watch cooking videos directly on YouTube
- 🎨 **Modern UI** — Clean, responsive design with smooth hover effects
- 📱 **Fully Responsive** — Works perfectly on desktop, tablet, and mobile
- 🔄 **Real-time Feedback** — Loading states and error handling
- 🎭 **Category Tags** — Quick visual category indicators
- ↩️ **Easy Navigation** — Back button to return to search results

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **HTML5** | Semantic markup structure |
| **CSS3** | Modern styling, animations, responsive design |
| **JavaScript (ES6+)** | Dynamic functionality & API integration |

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/razazaheer12/Recipe-Finder.git
   cd Recipe-Finder
   ```

2. **Open in browser**
   - Simply open `index.html` in your browser
   - Or use a live server extension in VS Code

## 📡 API Integration

This project uses **TheMealDB** — a free, open recipe API.

| Endpoint | Description |
|----------|-------------|
| `search.php?s={term}` | Search meals by name |
| `lookup.php?i={id}` | Get full meal details |

### Example Searches

Try searching for:
- `Chicken` 🍗
- `Pasta` 🍝
- `Pizza` 🍕
- `Burger` 🍔
- `Biryani` 🍚
- ` Curry` 🍛

## 🎯 Key Features Explained

### 1. Search Functionality
- Enter any meal name or keyword
- Press Enter or click Search button
- Results displayed in a responsive grid

### 2. Meal Cards
- Thumbnail image
- Meal title
- Category tag
- Hover effects with elevation

### 3. Meal Details View
- Large meal image
- Title and category
- Step-by-step cooking instructions
- Dynamic ingredients list (up to 20 ingredients)
- YouTube link for video tutorial
- Back button to return to results

### 4. Error Handling
- Empty search warning
- No results message
- Network error handling
